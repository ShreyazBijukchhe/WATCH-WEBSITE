# Watch E-Commerce Website

## Overview
A responsive product website for an online watch store. Built with **HTML5** and **CSS3** to showcase luxury watches with clean, modern design and responsive layout.

## Features
- ✅ **Responsive Design** - Mobile-first approach
- ✅ **Product Showcase** - Beautiful product cards with images
- ✅ **Product Filtering** - Filter by category, price, brand
- ✅ **Shopping Cart** - Add to cart functionality
- ✅ **Search Bar** - Find products quickly
- ✅ **Product Details** - Detailed specifications and pricing
- ✅ **User Reviews** - Customer testimonials
- ✅ **SEO Optimized** - Meta tags, structured data
- ✅ **Git Version Control** - Team collaboration ready

## Tech Stack
- **Frontend:** HTML5, CSS3
- **Icons:** Font Awesome
- **Layout:** CSS Grid & Flexbox
- **Responsive:** Mobile-first media queries
- **Version Control:** Git
- **Collaboration:** GitHub

## Pages Included
1. **Home** - Hero banner, featured products, promotions
2. **Products** - Complete product catalog with filters
3. **Product Details** - Individual watch specifications
4. **About Us** - Brand story and company info
5. **Contact** - Contact form and customer support
6. **Cart** - Shopping cart functionality

## Design Features
### Navigation

- High-quality product images

- Add to cart button

- Desktop: 1200px+
- Tablet: 768px - 1199px

## Installation
1. Clone repository
2. Open `index.html` in browser
3. No build tools required
